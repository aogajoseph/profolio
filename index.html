About Me

My name is Joseph Onyango Okoth. I'm a self-taught software developer learning to code at the Zero to Mastery academy on Udemy. My interest in computers began when I was a little boy and the passion to learn computer science and coding was sparked by a fierce curiousity to understand how computers really work.

I started my course on basic computer science and professional coding in February 2021. Since then, I've learnt just enough to write good HTML, CSS and JavaScript code. In addition, I've learnt how to make use of CSS libraries like Bootstrap and Fontawesome. Currently I'm looking forward to learning a JavaScript library, preferably React and scale on to JavaScript frameworks like Vue and Angular. I plan to gather lots of experience through front-end software development projects before I move on to back-end coding.

I can say with 100% confidence that in a few years to come, I'll have enough professional experience to be called a full-stack engineer. My long-term career goal is to work with more experienced engineers on different projects and eventually build my own SaaS (Software as a service) product. I find value in learning new things and improving my skills everyday.

Services

Below are some of the services I offer:

Web Development

I create fully responsive websites from scratch, with templates and on platforms such as Wix, Wordpress etc. I also handle the maintenance aspects therein such as publishing, programming and management.

Web Design

Applying my CSS skills, I am able to customize the look and feel of my websites to meet my clients' demands. I achieve this with appropriate UI/UX designs and semantic HTML/markup.

SEO Strategies

I apply Search Engine Optimization best practices such as proper title tags and meta descriptions to improve the positioning of my websites in organic search results hence improving the quality and quantity of traffic.

Debugging

Testing

Wireframing

I create wireframes and mock-ups to visualize the elements on my web pages, the content they will contain and their functionalities.
